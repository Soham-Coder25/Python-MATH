# Python-MATH
This repository contains all my math-related codes in Python.
It will have codes ranging from computation to simulations and functions.

In most of the "Do you want X" questions, 1 will be Yes, and other numbers, if unspecified, will be no.
In the input questions, if you enter an invalid number and an error shows up, simply restart the code, and enter numbers correctly.

For the function-based code, if you are inputing a function that is in the "math" module, then you must put "math." and then the appropriate function name.

Ex: instead of Cos(x) or cos(x), write math.cos(x)

Instead of Ceiling(x) or ceiling(x), write math.ceil(x)

I may update these rules later in the future, but for now, these remain the main rules. Any problems, suggestions or additions may be discussed in the discussions section.
